Modular Design

Case Study

A Drupal 8 theme integrated with Pattern Lab, a collection of tools for creating modular design systems.

Integrating pattern Lab enhances how designers, developers and clients collaborate, focusing on reusable UI elements and a centralized source for easy modification across patterns. Interactive updates mean components can be tested in the browser to optimize for responsiveness, accessibility and performance.

Design Highlights

Modular Design System

Two primary goals motivated the theme design. The client needed a design system that was easier to maintain and improved accessibility and performance.

Pattern Lab, an open-source tool for building UI design systems, was chosen as a way to streamline design, encourage a standardized approach and improve collaboration. The flexible approach reduces complexity through a system of UI components. A modular approach like Pattern Lab relies on smaller design components, combining them to create page sections and templates.

This approach is easier to maintain, eliminating time spent on different versions and edge cases. Instead, Pattern Lab provides a way to describe the stitched together patterns, helping designers and developers easily reference and update components. Knowing many designers and developers would contribute to the site, it was important to build a flexible and standardized design system.

Pattern Lab also provided a way to efficiently communicate with the client. Interactive wireframes are part of the workflow. Throughout the process, the client was able to interact with individual components and low-fidelity versions of final pages, supporting an iterative design process.

Responsive Design

Responsive websites focus on delivering a thoughtful user experience regardless of environment, bandwidth or specific user needs.

Quickly moving designs to the browser supports a mobile-first responsive approach. Module design is efficient, leaving more time for testing and optimizing components.

Pattern Lab provides a disco mode test for all the screen sizes!

Integrated Design System

The Drupal theme integrates pattern Lab, automatically updating the site when patterns are modified.

Drupal 8 improvements included a simplified theme layer and the introduction of the Twig templating language. This theme is built using the Gesso starter theme, a modern theme that leverages Drupal 8 improvements. Gesso refers to Drupal’s CSS best practices, using SMACSS to organize styles.

Gesso integrates Pattern Lab, automatically updating the theme when patterns are modified. This ensures consistency and saves time. An integrated component-based design system is efficient. The approach addresses the client's primary goals, making the the site easier to maintain and enabling more focused optimization.

Grunt compiles Sass and Pattern Lab files, watching them for changes. Flexbox was used for layout.

Content Strategy

Create Once
Publish Everywhere

Creating content once and publishing in multiple places widely disseminates content without overly burdening the creator.

After taking inventory of existing content and assessing organizational goals, a content strategy was created, informing the theme design. Featured content is promoted from the blog interface. Custom summaries and featured images are added and the appropriate version of the post displays when featured. Managing content in one place eliminates the need to maintain multiple sets of content. Relevant posts are featured with minimal effort and searchable archives are preserved.

Members with different roles contribute and curate featured content. The content strategy engages contributors while allowing privileged users to curate highly visible content. Based on the user’s permissions, instructions and resources for managing content are shown.

To engage as many members as possible, adding content is straightforward for users with limited access. Most volunteers are given Contributor accounts. When creating content, their admin interface has limited options beyond tagging the post and adding a custom summary. Once published, the post appears on designated pages.

A handful of trained users have advanced privileges, allowing them to promote posts and edit page content. Users with Editor accounts can promote any blog post, update page content and edit all content created by a Contributor. While Contributors provide the bulk of the of content, Editors strategically feature relevant articles and media throughout the site.
